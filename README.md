## ALH Website

First, run the development server:

```bash
pnpm install
# and then
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

before committing changes don't forget to do and resolve issue:

```bash
pnpm run lint
# and then
npm run lint
```
